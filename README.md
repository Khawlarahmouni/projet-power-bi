# projet-power-bi
Ce projet consiste à créer un outil d'analyse Make vs Buy avec Power BI. L'outil permet de comparer les coûts de fabrication interne et les coûts d'achat externe pour différents volumes de production, tout en intégrant des données internes de coûts pour une analyse plus approfondie.
Fonctionnalités principales :
Analyse des devis : Compare les coûts internes et externes selon divers scénarios de production.
Calcul des coûts : Permet de calculer les coûts pour différents volumes de production.
Intégration des données internes : Utilisation de données internes de fabrication pour des calculs plus précis.
Modèle extensible : Peut être enrichi avec des aspects liés à la chaîne d'approvisionnement, les délais de production, et d'autres coûts supplémentaires.
Technologies utilisées :
Power BI : Outil de visualisation des données pour créer des rapports interactifs.
Sources de données internes : Connexion avec des bases de données internes pour intégrer les données de fabrication.
Prérequis :
Avant de pouvoir exécuter ce projet, assurez-vous d'avoir les outils suivants installés :

Power BI Desktop : Pour ouvrir et interagir avec le rapport Power BI.

Clonez ou téléchargez ce dépôt depuis GitHub.
Ouvrez le fichier supply chail reportt.pbix dans Power BI Desktop.
